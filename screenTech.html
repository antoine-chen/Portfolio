Base de données "ScreenTech"

Description du projet :

Ce projet s’inscrit dans le cadre de la SAÉ S2.04 du semestre 2, qui vise à fournir une première approche complète de la conception, l’implémentation, l’administration et l’exploitation d’une base de données relationnelle. Le travail a été réalisé en trinôme.
La base de données modélise un site de vente en ligne spécialisé dans les écrans

Postgresql

SQL

Team work

VS code

Fonctionnalités principales

  • Gestion des utilisateurs : création, identification, rôles, commandes et paniers.
  • Catalogue de produits : avec détails techniques (résolution, dalle, dimensions), marque et fournisseur.
  • Suivi de l’état des commandes : en attente, expédié, validé, etc.

Apprentissage

  • Faire scénario textuel décrivant l’activité du site de vente d’écrans.

  • Le MCD et MLD

  • Des requêtes SQL complexes incluant jointures, sous-requêtes, agrégations, vues, etc.

  • Un script de création de la base de données PostgreSQL et insertion de données
  • Documentation du travail et organisation du travail en groupe
