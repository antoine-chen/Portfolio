Jeu de Plateau en Java

Description du projet :

Ce projet est un jeu de morpion amélioré développé en Java en binôme.
L'objectif du jeu est d'aligner cinq pions le plus possible dans un tableau à deux dimensions, en jouant tour à tour.

Java

JUnit

Intellij

Team work

Réflexion d'algo

Fonctionnalités principales

  • Mode 1v1 : Deux joueurs s'affrontent en tour par tour, utilisation du clavier pour saisir un pion puis affichage dans le terminal.
  • Interface en terminal avec affichage dynamique du plateau de jeu.
  • Un mode contre un bot : Choisissez entre trois niveaux de difficulté (facile, moyen, difficile), selon le niveau le bot a des fonctionnalités pour battre le joueur.

Apprentissage

  • Programmation en Java, je me suis occupé d'une partie partie java et les tests jUnit.
  • Travail en équipe et gestion de projet en binôme avec Github, nous avons du travailler pendant les cours et à la maison, donc toujours avoir le code propre pour pourvoir travailler.
  • Il fallait concevoir et réflechir sur les algorithmiques pour que l'IA puisse jouer contre un joueur.
