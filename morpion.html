Jeu de Plateau en Java

Description du projet :

Ce projet est un jeu de morpion amélioré développé en Java en binôme.
L'objectif du jeu est d'aligner cinq pions le plus possible dans un tableau à deux dimensions, en jouant tour à tour.

Java

JUnit

Intellij

Team work

Réflexion d'algo

Fonctionnalités principales

  • Mode 1v1 : Deux joueurs s'affrontent en tour par tour.
  • Mode contre un bot : Choisissez entre trois niveaux de difficulté (facile, moyen, difficile).
  • Interface en terminal avec affichage dynamique du plateau de jeu.

Apprentissage

  • Programmation en Java
  • Travail en équipe et gestion de projet en binôme avec Github
  • Conception et réflexion algorithmique pour l'IA du bot
