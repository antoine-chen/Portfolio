Echecs

Description du projet :

Dans le cadre de ce projet collaboratif, j'ai contribué à la réalisation d'une application d'échecs en Java. Mon rôle a consisté à concevoir et développer les jeux de tests ainsi qu'à participer à la réflexion algorithmique.

Java

JUnit

Intellij

Team work

Réflexion d'algo

Fonctionnalités principales

  • Mode 1v1 : Un jeu d'échecs entièrement jouable, doté d'une interface.
  • Mécanismes de validation des mouvements et gestion complète des règles des échecs.

Apprentissage

  • Jeu de tests : Élaboration de tests unitaires pour assurer la fiabilité du code et la stabilité du jeu.
  • Travail en équipe, debug problème de colision ou juste des bugs qui ne respectent pas les règles du jeu
  • Réflexion algorithmique pour pourvoir faire un echec et mat.
