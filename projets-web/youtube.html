Youtube

Description du projet :

L’objectif de ce projet était de concevoir un site web inspiré de YouTube (partiellement), avec plusieurs fonctionnalités clés : lecture de vidéos intégrée, navigation par playlists et affichage des miniatures.

Le site devait également être responsive. J'ai réalisé ce projet individuellement, en utilisant HTML, CSS pour structurer l’interface et gérer l’interaction avec les contenus. Ce projet nous a permis de mettre en pratique des compétences en intégration front-end..

Présentation du site

  • Affichage des vidéos recommandées sur la page d'accueil.
  • Barre de navigation avec logo YouTube, barre de recherche et icônes utilisateur
  • Sidebar avec différentes catégories (Tendances, Abonnements, Bibliothèque, etc.)
  • Responsive youtube sur telephone tablette et ordinateur.

Apprentissage

  • Organisation et structuration du HTML/CSS
  • Utilisation des flexbox et grid pour une mise en page dynamique
  • Debug sur des problèmes qui sont liées au CSS
  • Responsive niveau
